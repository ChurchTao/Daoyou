# 游戏UI重构计划

## 1. 重构目标

根据设计规范文档，将现有游戏UI重构为**以文字为核心、水墨风格、移动端优先、沉浸感强**的界面，确保视觉风格、交互模式、布局结构等符合统一规范。

## 2. 重构范围

- 全局样式和布局
- 首页（主界面）
- 角色详情页
- 储物袋界面
- 技能/神通界面
- 炼器/顿悟/奇遇界面
- 挑战界面
- 战报界面
- 导航系统

## 3. 重构步骤

### 3.1 全局样式重构

1. **完善CSS变量**：确保颜色、字体等统一
2. **添加风格元素**：
   - 分隔线样式（`────────────────` 或 `☯☯☯☯☯`）
   - 状态图标样式（❤️、⚡️、☯、🌿等Unicode符号）
   - 修仙语境的加载提示、空状态、错误提示
3. **优化移动端适配**：
   - 单列布局
   - 操作区域集中在屏幕下半部
   - 响应式字体大小

### 3.2 首页重构

1. **重新设计布局**：
   - 顶部固定角色状态栏（姓名、境界、气血、灵力）
   - 天机模块（今日宜忌）
   - 快捷入口（六宫格按钮）
   - 近期战绩
   - 底部引文
2. **实现底部固定导航栏**：
   - 首页、储物、神通、天机榜
   - 当前页高亮显示

### 3.3 角色详情页重构

1. **重新设计布局**：
   - 道我真形标题
   - 角色基本信息
   - 先天命格
   - 根基属性
   - 当前所御法宝
   - 所修神通
2. **添加导航链接**：
   - 前往储物袋更换装备
   - 闭关顿悟新神通
   - 返回主界/推演战力

### 3.4 储物袋界面重构

1. **重新设计布局**：
   - 法宝列表，显示已装备状态
   - 装备/卸下按钮
   - 整理法宝功能
2. **优化交互**：
   - 每件装备下方只有一个操作按钮
   - 装备描述用"｜"分隔属性

### 3.5 技能/神通界面重构

1. **重新设计布局**：
   - 神通列表，显示新悟标记
   - 替换按钮
   - 闭关顿悟新神通入口
2. **优化交互**：
   - 显示已达上限提示（如"3/3"）

### 3.6 炼器/顿悟/奇遇界面重构

1. **统一输入界面设计**：
   - 提供示例降低用户输入门槛
   - 多行文本输入框，占屏幕40%高度
   - 按钮右对齐
2. **优化交互**：
   - 清晰的取消/开始按钮
   - 修仙语境的提示文字

### 3.7 挑战界面重构

1. **重新设计布局**：
   - 天骄榜标题
   - 排行榜列表，显示排名、名称、战力
   - 挑战按钮
   - 刷新榜单功能
2. **优化交互**：
   - 高亮显示当前角色
   - 每行只保留一个操作按钮

### 3.8 战报界面重构

1. **优化战报展示**：
   - 全屏展示AIGC生成的战报
   - 不显示数值过程，只保留小说式描写
   - 增强水墨风格排版
2. **优化交互**：
   - 再战/返回主界/分享战报按钮
   - 响应式布局

### 3.9 导航系统优化

1. **实现全局导航**：
   - 底部固定导航栏（仅主界面显示）
   - 所有子页面左上角返回按钮
2. **优化导航体验**：
   - 用文字而非图标，避免歧义
   - 当前页高亮显示

## 4. 技术实现策略

1. **组件化设计**：将UI拆分为可复用组件，确保一致性
2. **状态管理**：使用React状态管理当前页面状态
3. **假数据填充**：对于未实现的功能点，使用假数据填充或硬编码方式进行临时占位
4. **明确标记**：对未实现的功能点做好明确标记，以便后续开发完善
5. **响应式设计**：使用CSS媒体查询实现移动端和PC端适配

## 5. 测试与优化

1. **视觉一致性检查**：确保所有页面风格统一
2. **基础交互测试**：确保所有交互功能正常
3. **移动端适配测试**：确保在手机上有良好的体验
4. **性能优化**：确保页面加载和交互流畅

## 6. 预期成果

- 所有页面符合设计规范要求
- 视觉风格统一，具有水墨意境
- 移动端优先，操作便捷
- 沉浸感强，符合修仙语境
- 响应式设计，适配PC和移动端

## 7. 开发顺序

1. 全局样式重构
2. 首页重构
3. 角色详情页重构
4. 储物袋界面重构
5. 技能/神通界面重构
6. 炼器/顿悟/奇遇界面重构
7. 挑战界面重构
8. 战报界面重构
9. 导航系统优化
10. 测试与优化

这个计划将确保游戏UI按照设计规范进行系统性重构，实现以文字为核心、水墨风格、移动端优先、沉浸感强的界面效果。
