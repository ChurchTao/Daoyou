# 凌霄剑宗 V4.1 平衡基准

## 本轮数值调整

- 不动藏锋后发：`2.60物攻 → 2.20物攻`。
- 听雷节点后发：`3.10物攻 → 2.70物攻`。
- 原因：后发在 V4.1 改为登记后不可打断且必然命中，旧倍率仍包含“可能被控制取消或闪避”的风险补偿。
- 未修改全局气血公式。测试期间验证过简单放大气血会显著放大重剑持续护盾与减伤收益，因此不以全局公式掩盖流派循环问题。

## 三境界固定种子基准

以下数据使用实际境界/阶段属性预算、相同心法等级、无装备和灵根优势，每项 100 局并交换先手位置。格式为“快剑胜率 / 平均回合”。

| 代表对局 | 筑基初期 | 金丹圆满 | 化神圆满 |
|---|---:|---:|---:|
| 追风快剑 vs 后发重剑 | 44% / 6.32 | 60% / 5.05 | 20% / 4.43 |
| 连势快剑 vs 开天重剑 | 7% / 5.63 | 7% / 4.68 | 9% / 3.39 |
| 回燕快剑 vs 守山重剑 | 0% / 10.95 | 0% / 9.96 | 0% / 7.67 |

当前基准没有达到计划中的 45%–55% 代表对局胜率、35%–65% 常见交叉克制和 8–12 回合要求。机制、日志和来源归因已经足以稳定测量问题，但继续调倍率前需要先重新校准三套推荐装配与战术选招，否则数值调整会同时掩盖“回燕装配缺少稳定产势/输出”和“开天装配循环效率过高”两个策略问题。
